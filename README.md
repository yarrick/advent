## Solutions in haskell to adventofcode.com puzzles

To build, run `make` followed by the two-digit number for the day.
To get the answer, execute the binary and send the input to stdin.

### Required cabal libraries

* arithmoi
* cryptonite
* matrix
* sequence
* vector
